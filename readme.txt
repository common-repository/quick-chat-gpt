Add a GPT-3 powered chatbot to the WordPress admin panel.

== Description ==

Quick Chat GPT adds a GPT-3 powered chatbot to the WordPress admin panel. The chatbot can assist with a variety of tasks, such as answering questions about your website, providing tutorials, and more.

To use the plugin, simply activate it and an edit icon will appear on the top right corner of the admin panel. Clicking on the icon will open the chatbot interface where you can start a conversation.

You can also customize the chatbot settings by going to the Quick Chat GPT settings page in the WordPress admin.

== Installation ==

Upload the plugin files to the /wp-content/plugins/quick-chat-gpt directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
An edit icon will appear on the top right corner of the admin panel. Clicking on the icon will open the chatbot interface where you can start a conversation.
You can also customize the chatbot settings by going to the Quick Chat GPT settings page in the WordPress admin.
== Frequently Asked Questions ==

= How does the chatbot work? =

The chatbot uses the GPT-3 API from OpenAI to generate responses to your questions and commands. The plugin sends your input to the API and receives a response, which is then displayed in the chatbot interface.

= Is an API key required? =

Yes, an API key from OpenAI is required to use the plugin. You can sign up for an API key on the OpenAI website at https://platform.openai.com/account/api-keys.
